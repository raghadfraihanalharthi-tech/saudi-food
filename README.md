Saudi Cuisine Website:
A group project for the Web Development course at university.
About the Project
This website is about traditional Saudi cuisine. It was built using HTML only as part of a university assignment. The website includes information about famous Saudi dishes, their history, recipes, and contact details.
Pages

index.html - Home page with an overview of the website and featured dishes
basic.html - History of Saudi cuisine using basic HTML structure with div elements
semantic.html - Popular Saudi dishes using HTML5 semantic elements
recipes.html - Step-by-step recipes for traditional Saudi dishes
contact.html - Contact information, FAQ, and a quote about Saudi food

Technologies Used

HTML 5 only
No CSS or JavaScript

Key Features:

Basic HTML structure page using div elements
Semantic HTML5 page using header, nav, main, section, article, and footer
Relative paths connecting all pages
Special elements including blockquote, address, and details
Images for each dish

My Role: Project Lead
I was responsible for leading and managing the full project from start to finish. My contributions included choosing the project topic and planning the overall structure, dividing the work between the team members and assigning each one a specific page, building the home page myself using basic HTML structure with div elements, setting the unified footer that all pages use to keep the website consistent, reviewing every page submitted by the team and correcting any issues, managing the folder structure and making sure all files and images were named and placed correctly, and packaging the final ZIP file for submission.
